    In a simple single-page app, the template consists of HTML, CSS, and angular directives contained in just one HTML file (usually code). In a more complex app, you can display multiple views within one main page using "partials", which are segments of template located in separate HTML files. You "include" the partials in the main page using the $route service in conjunction with the ngView directive.
