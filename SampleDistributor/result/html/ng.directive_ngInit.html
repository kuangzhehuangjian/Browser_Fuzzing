code
directive in module code

Description

The code directive allows you to evaluate an expression in the current scope.

The only appropriate use of code is for aliasing special properties of code, as seen in the demo below. Besides this case, you should use controllers rather than code to initialize values on a scope.
Note: If you have assignment in code along with code, make sure you have parenthesis for correct precedence:
