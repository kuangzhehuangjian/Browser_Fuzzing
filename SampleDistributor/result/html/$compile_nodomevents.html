Description

This error occurs when one tries to create a binding for event handler attributes like code, code, code, etc.

There is no practical value in binding to these attributes and doing so only exposes your application to security vulnerabilities like XSS. For these reasons binding to event handler attributes (all attributes that start with code and code attribute) is not supported.
