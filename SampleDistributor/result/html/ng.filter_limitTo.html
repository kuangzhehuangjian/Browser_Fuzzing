Description

Creates a new array or string containing only a specified number of elements. The elements are taken from either the beginning or the end of the source array or string, as specified by the value and sign (positive or negative) of code.

Usage

pre

Parameters

ParamTypeDetails
inputArraystring

Source array or string to be limited.

limitstringnumber

The length of the returned array or string. If the code number is positive, code number of items from the beginning of the source array/string are copied. If the number is negative, code number of items from the end of the source array/string are copied. The code will be trimmed if it exceeds code

Returns

Array|string

A new sub-array or substring of length code or less if input array had less than code elements.
