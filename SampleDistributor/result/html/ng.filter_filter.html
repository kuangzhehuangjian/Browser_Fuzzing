filter in module code

Description

Selects a subset of items from code and returns it as a new array.

Usage

In HTML Template Binding

code

In JavaScript

code

Parameters

ParamTypeDetails
arrayArray

The source array.

expressionstringObjectfunction()

The predicate to be used for selecting items from code.

Can be one of:

  • code: The string is evaluated as an expression and the resulting value is used for substring match against the contents of the code. All strings or objects with string properties in code that contain this string will be returned. The predicate can be negated by prefixing the string with code.

  • code: A pattern object can be used to filter specific properties on objects contained by code. For example code predicate will return an array of items which have property code containing "M" and property code containing "1". A special property name code can be used (as in code) to accept a match against any property of the object. That's equivalent to the simple substring match with a code as described above.

  • code: A predicate function can be used to write arbitrary filters. The function is called for each element of code. The final result is an array of those elements that the predicate returned true for.

comparatorfunction(actual, expected)trueundefined

Comparator which is used in determining if the expected value (from the filter expression) and actual value (from the object in the array) should be considered a match.

Can be one of:

  • code: The function will be given the object value and the predicate value to compare and should return true if the item should be included in filtered result.

  • code: A shorthand for code. this is essentially strict comparison of expected and actual.

  • code: A short hand for a function which will look for a substring match in case insensitive way.
