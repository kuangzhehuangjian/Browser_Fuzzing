Description

This error occurs when code tries to process a directive that specifies the code option in a code, but the required directive controller is not present on the current DOM element (or its ancestor element, if code was specified).

To resolve this error ensure that there is no typo in the required controller name and that the required directive controller is present on the current element.

If the required controller is expected to be on a ancestor element, make ensure that you prefix the controller name in the code definition with code.

If the required controller is optionally requested, use code or code to specify that.
