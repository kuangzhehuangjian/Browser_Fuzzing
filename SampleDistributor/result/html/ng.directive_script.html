directive in module code

Description

Load the content of a code element into code, so that the template can be used by code, code, or directives. The type of the code element must be specified as code, and a cache name for the template must be assigned through the element's code, which can then be used as a directive's code.

Usage

This directive can be used as custom element, but be aware of IE restrictions.
