code
service in module code

Description

The code service is a service that abstracts the messier details of hold-and-drag swipe behavior, to make implementing swipe-related directives more convenient.

Requires the code module to be installed.

code is used by the code and code directives in code, and by code in a separate component.

Usage

The code service is an object with a single method: code. code takes an element which is to be watched for swipes, and an object with four handler functions. See the documentation for code below.

Methods

  • bind()

    The main method of code. It takes an element to be watched for swipe motions, and an object containing event handlers.

    The four events are code, code, code, and code. code, code, and code receive as a parameter a coordinates object of the form code.

    code is called on either code or code. After this event, code is watching for code or code events. These events are ignored until the total distance moved in either dimension exceeds a small threshold.

    Once this threshold is exceeded, either the horizontal or vertical delta is greater. - If the horizontal distance is greater, this is a swipe and code and code events follow. - If the vertical distance is greater, this is a scroll, and we let the browser take over. A code event is sent.

    code is called on code and code after the above logic has determined that a swipe is in progress.

    code is called when a swipe is successfully completed with a code or code.

    code is called either on a code from the browser, or when we begin scrolling as described above.
