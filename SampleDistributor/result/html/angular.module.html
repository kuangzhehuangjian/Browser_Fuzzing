API in module code

Description

The code is a global place for creating, registering and retrieving Angular modules. All modules (angular core or 3rd party) that should be available to an application must be registered using this mechanism.

When passed two or more arguments, a new module is created. If passed only one argument, an existing module (the name passed as the first argument to code) is retrieved.

Module

A module is a collection of services, directives, filters, and configuration information. code is used to configure the code.
