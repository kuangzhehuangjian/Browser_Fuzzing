code
API in module code

Description

Invokes the code function once for each item in code collection, which can be either an object or an array. The code function is invoked with code, where code is the value of an object property or an array element and code is the object property key or array element index. Specifying a code for the function is optional.

It is worth noting that code does not iterate over inherited properties because it filters using the code method.
