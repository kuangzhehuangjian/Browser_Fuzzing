Description

Determines if two objects or two values are equivalent. Supports value types, regular expressions, arrays and objects.

Two objects or values are considered equivalent if at least one of the following is true:

  • Both objects or values pass code comparison.
  • Both objects or values are of the same type and all of their properties are equal by comparing them with code.
  • Both values are NaN. (In JavaScript, NaN == NaN => false. But we consider two NaN as equal)
  • Both values represent the same regular expression (In JavasScript, /abc/ == /abc/ => false. But we consider two regular expressions as equal when their textual representation matches).

During a property comparison, properties of code type and properties with names that begin with code are ignored.

Scope and DOMWindow objects are being compared only by identify (code).
