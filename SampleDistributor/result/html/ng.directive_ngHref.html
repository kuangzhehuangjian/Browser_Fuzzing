code
directive in module code

Description

Using Angular markup like code in an href attribute will make the link go to the wrong URL if the user clicks it before Angular has a chance to replace the code markup with its value. Until Angular replaces the markup the link will be broken and will most likely return a 404 error.

The code directive solves this problem.

The wrong way to write it: 

pre

The correct way to write it: 

pre
