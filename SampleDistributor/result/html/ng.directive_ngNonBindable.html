code
directive in module code

Description

The code directive tells Angular not to compile or bind the contents of the current DOM element. This is useful if the element contains what appears to be Angular directives and bindings but which should be ignored by Angular. This could be the case if you have a site that displays snippets of code, for instance.

Usage

as attribute
pre
as class
pre

Directive info

  • This directive executes at priority level 1000.
