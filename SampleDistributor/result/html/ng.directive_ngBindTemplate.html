code
directive in module code

Description

The code directive specifies that the element text content should be replaced with the interpolation of the template in the code attribute. Unlike code, the code can contain multiple code code expressions. This directive is needed since some HTML elements (such as TITLE and OPTION) cannot contain SPAN elements.
