Description

This error occurs when you configure the code service in the html5 mode, specify a base url for your application via code element and try to update the location with a path that doesn't match the base prefix.

To resolve this issue, please check the base url specified via the code tag in the head of your main html document, as well as the url that you tried to set the location to.
