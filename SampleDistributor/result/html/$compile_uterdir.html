error in component code 

pre

Description

This error occurs when using multi-element directives and a code attribute fails to form a matching pair with a corresponding code attribute. A code should have a matching code on a sibling node in the DOM. For instance,

pre

is a valid example.

This error can occur in several different ways. One is by leaving out the code attribute, like so:

pre

Another is by nesting a code inside of code, or vice versa:

pre

To avoid this error, make sure each code you use has a matching code on a sibling node in the DOM.
