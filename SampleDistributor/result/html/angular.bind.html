code
API in module code

Description

Returns a function which calls function code bound to code (code becomes the code for code). You can supply optional code that are prebound to the function. This feature is also known as partial application, as distinguished from function currying.

Usage

pre

Parameters

ParamTypeDetails
selfObject

Context which code should be evaluated in.

fnfunction()

Function to be bound.

args...*

Optional arguments to be prebound to the code function call.

Returns

function()

Function that wraps the code with all the specified bindings.
