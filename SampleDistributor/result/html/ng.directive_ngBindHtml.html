code
directive in module code

Description

Creates a binding that will innerHTML the result of evaluating the code into the current element in a secure way. By default, the innerHTML-ed content will be sanitized using the $sanitize service. To utilize this functionality, ensure that code is available, for example, by including ngSanitize in your module's dependencies (not in core Angular.) You may also bypass sanitization for values you know are safe. To do so, bind to an explicitly trusted value via code. See the example under code.

Note: If a code service is unavailable and the bound value isn't explicitly trusted, you will have an exception (instead of an exploit.)
